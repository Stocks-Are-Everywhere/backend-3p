### 📍 프로젝트 개요
국내 주식을 기반으로 한 모의 투자 서비스입니다. 사용자는 실전 투자와 유사한 환경에서 주식 매매를 연습하고, 데이터 기반 분석을 통해 투자 전략을 수립할 수 있습니다.
